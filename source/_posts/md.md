#### MD 常用语法

> Headers 标题：
> `#{n} H{n} n:1-6`

> Emphasis 文本强调
> _斜体_ **加粗** **_粗斜体_** <u>下划线</u>

> Lists 列表：

- 无序列表

1. 第一行
2. 第二行

<!--more-->

> Links 连接（title 为可选项）：
> [这里是链接](https://www.google.com)
> [link text]: http://www.reddit.com

> Email ：
> <example@example.com>

> Images :


```css

  font-size: 12px;
}
```

```javascript
var s = "JavaScript syntax highlighting";
alert(s);
```

```php

    ?>
```

```python
    s = "Python syntax highlighting"
    print s
```
